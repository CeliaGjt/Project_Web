Bienvenue

Dans ce projet, l’administration des ChatBots se fait à travers une API REST. Ce site permet d'administrer un ChatBot, c'est à dire de le créer, le modifier (en changeant sa personnalité par exemple) et le supprimer... Afin de communiquer avec les chatbots, plusieurs interfaces de communication sont à votre disposition : la première est disponible sur ce site à l'onglet plateforme de communication , la deuxième est discord (le bot peut être directement connecté à un bot discord grâce à son token).
